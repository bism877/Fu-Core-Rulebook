# FU Core Rulebook

A Foundry VTT module for the **Project FU system**, containing structured compendiums inspired by the Fabula Ultima Core Rulebook.

This module provides organized reference content for gameplay, including creatures and rare items.

---

## 📦 Contents

- Core Rulebook Bestiary (Actors)
- Core Rulebook Rare Items (Items)

All content is organized into a dedicated compendium folder inside Foundry.

---

## 🧩 Requirements

- Foundry VTT v13+
- Project FU system installed

---

## ⚙️ Installation


### Option 1 — Manual Installation

1. Download the module
2. Extract it into your Foundry `modules/` folder:

3. Restart Foundry
4. Enable the module in your world settings

---

## 📁 Structure

The module automatically creates a compendium folder: FU Core Rulebook
