# 🌙 FU Core Rulebook

A homebrew module for Foundry VTT focused on **Fabula Ultima / Project FU**, providing compendiums and structure to support gameplay, testing, and automation.

---

## ✨ Contents

This module includes:

- 📚 Core Rulebook Bestiary compendium  
- 💎 Core Rulebook Rare Items compendium  
- ⚙️ Ready-to-use structure for future expansion  
- 🧠 Base setup for Project FU automation  

---

## 📦 Installation

### 🔗 Foundry VTT (recommended)

In Foundry VTT:

1. Go to **Add-on Modules**
2. Click **Install Module**
3. Paste the manifest URL:

https://github.com/bism877/Fu-Core-Rulebook/releases/latest/download/module.json


---

### 🧭 Manual Installation (Windows / AppData)

#### 📍 Find your Foundry User Data folder

Default location on Windows:
C:\Users<YOUR_USERNAME>\AppData\Local\FoundryVTT\Data\modules\

You can also open it directly from Foundry:

> Settings → Show Data Folder

---

#### 📦 Install steps

1. Download the module:
https://github.com/bism877/fu-core-rulebook/archive/refs/heads/main.zip

2. Extract the ZIP file

3. Move the folder into: .../Data/modules/

4. Make sure the final structure is:
Data/modules/Fu-Core-Rulebook/module.json


5. Restart Foundry VTT
